д»Җд№ҲжҳҜServer Side Includes

жҲ‘д»¬жқҘзңӢзңӢз»ҙеҹәзҷҫз§‘жҳҜжҖҺд№Ҳи§ЈйҮҠзҡ„:

жңҚеҠЎеҷЁз«ҜеҶ…еөҢпјҲServer Side IncludesпјҢдәҰз®Җз§°дёәSSIпјүжҳҜдёҖз§ҚеӨ§еӨҡж•°д»…еә”з”ЁдәҺдә’иҒ”зҪ‘дёҠзҡ„з®ҖеҚ•и§ЈйҮҠжҖ§жңҚеҠЎеҷЁз«Ҝи„ҡжң¬иҜӯиЁҖгҖӮ
SSIжңҖеёёи§Ғзҡ„з”Ёжі•жҳҜе°ҶдёҖдёӘжҲ–еӨҡдёӘж–Үд»¶зҡ„еҶ…е®№еҢ…еҗ«еңЁзҪ‘йЎөжңҚеҠЎеҷЁзҡ„йЎөйқўдёҠгҖӮдҫӢеҰӮпјҢдёҖеј еҢ…еҗ«жҜҸж—ҘжҠҘд»·зҡ„йЎөйқўеҸҜд»ҘйҖҡиҝҮдёӢйқўдёҖж®өд»Јз Ғе°ҶжҠҘд»·еҚ•еҢ…еҗ«еңЁйЎөйқўдёӯпјҡ

SSI з®Җз§°д»Јз ҒзүҮж®өпјҢйҖҡиҝҮзү№ж®Ҡзҡ„иҜӯжі•зі–пјҢжңҚеҠЎеҷЁеңЁе“Қеә”зҡ„ж—¶еҖҷйҒҮеҲ°зӣёе…іжҢҮд»ӨпјҢдјҡеҸ‘иө·еҸҰеӨ–зҡ„иҜ·жұӮпјҲsubrequestпјүеҺ»иҜ·жұӮеҜ№еә”зҡ„д»Јз ҒзүҮж®өд№ӢеҗҺеөҢе…ҘеҲ°зҲ¶йЎөйқўдёӯпјҢд»ҺиҖҢе®һзҺ°йЎөйқўеңЁжңҚеҠЎз«ҜжӢјжҺҘ

SSI зҡ„иҝҗдҪңжңәеҲ¶

еҰӮдҪ•дҪҝз”ЁServer Side Includes

дҪҝз”Ёnginx ssiжҢҮд»ӨеҚіеҸҜејҖеҗҜжңҚеҠЎ

жҲ‘д»¬д»ҘnginxдёәдҫӢе®һзҺ°дёҖдёӘssi жңҚеҠЎ, дҪҝз”Ёnginx д»ЈзҗҶеҲ° nodejs(иҝҷйҮҢжҲ‘д»¬з§°дёәupstreamжңҚеҠЎ), з„¶еҗҺеңЁnginxдёҠйқўејҖеҗҜssi жңҚеҠЎгҖӮ

й“ҫи·ҜеҰӮдёӢпјҡ

nginxи®ҫзҪ®д»ЈзҗҶ

nginx и®ҫзҪ®д»ЈзҗҶе№¶ејҖеҗҜssi

upstream жңҚеҠЎзӣҙжҺҘе“Қеә”htmlеҶ…е®№

з„¶еҗҺжү“ејҖйЎөйқў

жҳҜдёҚжҳҜеҫҲз®ҖеҚ•пјҹ

ејӮеёёзҡ„еҮәзҺ°

йӮЈд№ҲпјҢеҜ№дәҺSSIзҡ„дҪҝз”Ёж•°йҮҸпјҢжҳҜеҗҰжңүдёҠйҷҗе‘ўпјҹи®©жҲ‘д»¬еҫҖйЎөйқўйҮҢйқўеҶҚж·»еҠ SSI

дҝ®ж”№еҰӮдёӢ

и®ҝй—®з»“жһң

oops еҸ‘з”ҹдәҶд»Җд№ҲдәӢпјҹ

еңЁnginx зҡ„ error logйҮҢпјҢжҲ‘д»¬еҸ‘зҺ°й—®йўҳжүҖеңЁ

worker_connections are not enough while connecting to upstream

е°Ҹз»“

иҝҷйҮҢдҪ“зҺ°дәҶssi зҡ„з¬¬дёҖдёӘзү№жҖ§: subrequest еҗҢж ·д№ҹдјҡж¶ҲиҖ—workerзҡ„ж•°йҮҸ
жүҖд»ҘпјҢssiеҗҢж ·д№ҹдјҡи·ҹжҲ‘д»¬зҡ„зҲ¶иҜ·жұӮжҠўеҚ workerиө„жәҗпјҢжҜҸдёӘйЎөйқўзҡ„SSIж•°йҮҸд№ҹйңҖиҰҒдёҘж јиҝӣиЎҢйҷҗеҲ¶гҖӮ

еҜ№upstreamзҡ„жҢ‘жҲҳ

ж—ўз„¶зҲ¶иҜ·жұӮдёҺSSIйғҪдјҡеҚ з”Ёworkerж•°йҮҸзҡ„иҜқпјҢжҲ‘д»¬еҫҲе®№жҳ“жғіеҲ°пјҢдҪҝз”ЁSSIзҡ„жҠҖжңҜпјҢеҜ№еҗҺз«Ҝзҡ„жөҒйҮҸдјҡйҖ жҲҗж”ҫеӨ§

е®һйӘҢ

еҒҮи®ҫжҲ‘д»¬еңЁйЎөйқўйҮҢйқўиҜ·жұӮmдёӘSSIпјҢиҖҢupstream serverзҡ„жңҖеӨ§ж”Ҝж’‘иҜ·жұӮж•°дёәn, дё”m > n, иҝҷдёӘж—¶еҖҷдјҡеҸ‘з”ҹд»Җд№ҲдәӢпјҹ

жҲ‘д»¬жҢүз…§дёӢйқўзҡ„иЎЁж јеҜ№жңҚеҠЎиҝӣиЎҢж”№йҖ 

иҜ·жұӮж•° server-side include file upstream max connection
1 10 3

еҰӮдёҠйқўзҡ„иҜ·жұӮпјҢжҲ‘д»¬еңЁдёҖдёӘзҲ¶ж–Үд»¶йҮҢйқўи®ҫзҪ®10дёӘssiпјҢиҖҢupstreamжңҚеҠЎжҲ‘д»¬и®ҫзҪ®йҷҗжөҒ3дёӘиҜ·жұӮпјҢиҜ·жұӮзҲ¶ж–Үд»¶д№ӢеҗҺзңӢзңӢдјҡеҸ‘з”ҹд»Җд№Ҳпјҡ

еңЁhtmlж–Үд»¶дёҠи®ҫзҪ®еӨҡдёӘSSIзүҮж®ө

еҜ№upstream serverеўһеҠ йҷҗжөҒйҖ»иҫ‘пјҢдҪҝеҫ—SSIиҜ·жұӮи§ҰеҸ‘йҷҗжөҒ

е“Қеә”з»“жһң

е°Ҹз»“

дҪҝз”ЁSSI дјҡеҜ№жәҗз«ҷиҜ·жұӮзҡ„ж•°йҮҸж”ҫеӨ§пјҢеҜ№жәҗз«ҷдә§з”ҹжӣҙеӨ§зҡ„еҺӢеҠӣжҢ‘жҲҳ

еҗҢдёҖдёӘиҜ·жұӮ, SSI subrequestжҳҜе№¶иЎҢиҝҳжҳҜдёІиЎҢзҡ„

жңүжІЎжңүеҸ‘зҺ°пјҢдёҠиҝ°зҡ„SSI subrequest пјҢеүҚйқўзҡ„иҜ·жұӮйғҪжҳҜеҘҪзҡ„пјҢиҖҢиў«йҷҗжөҒзҡ„ssi subrequest пјҢйғҪжҳҜжҺ’еңЁеҗҺйқўгҖӮ

иҝҷйҮҢжҲ‘д»¬жңүдёӘз–‘й—®пјҡSSIзҡ„иҜ·жұӮжҳҜдёІиЎҢзҡ„еҗ—пјҹ

е…¶е®һд»ҺеүҚж–Үworker connectionйҷҗеҲ¶жқҘзңӢпјҢиҝҷдёӘиЎҢдёәеә”иҜҘжҳҜеҖҫеҗ‘дәҺе№¶иЎҢиҜ·жұӮгҖӮеҰӮжһңжҳҜдёІиЎҢиҜ·жұӮзҡ„иҜқпјҢйӮЈд№Ҳworker connectionеә”иҜҘжІЎжңүйӮЈд№Ҳе®№жҳ“е°ұж¶ҲиҖ—е®ҢгҖӮжҲ‘д»¬жҺЁжөӢпјҡ

  1. е…ҲиҜ·жұӮе®ҢзҲ¶иҜ·жұӮ
  2. и§Јжһҗhtmlд№ӢеҗҺпјҢж”¶йӣҶжүҖжңүзҡ„еӯҗиҜ·жұӮ
  3. е№¶иЎҢжүҖжңүзҡ„еӯҗиҜ·жұӮгҖӮ

дёҠйқўд»…д»…еҸӘжҳҜзҢңжөӢпјҢзңҹе®һжғ…еҶөдёӢпјҢеҲ°еә•жҳҜе№¶иЎҢиҜ·жұӮпјҢиҝҳжҳҜдёІиЎҢиҜ·жұӮпјҢиҝҳжҳҜиҰҒе®һйӘҢдәҶжүҚзҹҘйҒ“гҖӮ

и®ҫи®Ўе®һйӘҢ

еҜ№дәҺжҲ‘д»¬зҡ„node serverпјҢжҲ‘д»¬иҝӣиЎҢеҰӮдёӢж”№йҖ пјҢи®ҫзҪ®еҠЁжҖҒи·Ҝз”ұ

еҰӮжһңиҜ·жұӮupstream жңҚеҠЎпјҢ/dynamic/sleep/1000 еҲҷдјҡзӯүеҫ…1sжүҚе“Қеә”пјҢ/dynamic/sleep/2000 еҲҷжҳҜ2sжүҚе“Қеә”

жҺҘзқҖи®ҫзҪ®жҲ‘д»¬зҡ„дё»html

еҰӮжһңиҜ·жұӮindex.htmlж–Үд»¶пјҢеҲҷдјҡи§ЈжһҗеҲ°3дёӘserver-side include file, 3дёӘеӯҗиҜ·жұӮеҠ иҪҪж—¶й—ҙеҲҶеҲ«дёә10s, 8s, 5s.

еҰӮжһңжҳҜдёІиЎҢпјҢйӮЈд№Ҳж—¶й—ҙеә”и¶…иҝҮ 10 + 8 + 5 = 23s;
еҰӮжһңжҳҜе№¶иЎҢпјҢж—¶й—ҙеә”иҜҘжҺҘиҝ‘10s

иҜ·жұӮйЎөйқўпјҢжҲ‘д»¬зңӢзңӢдјҡеҸ‘з”ҹд»Җд№Ҳ
еңЁжөҸи§ҲеҷЁиҜ·жұӮзҡ„timelimeдёҠжҲ‘д»¬еҸҜд»ҘеҫҲжё…жҘҡзҡ„зңӢеҲ°пјҢиҜ·жұӮзҡ„ж—¶й—ҙзәҰдёә10s

еңЁnode serverзҡ„ж—Ҙеҝ—

жҲ‘д»¬еҸҜд»ҘзңӢеҲ°пјҡ

  1. жңҚеҠЎз«ҜжҺҘеҸ—еҲ°дәҶзҲ¶иҜ·жұӮ, е№¶жҲҗеҠҹе“Қеә”
  2. еҗҢж—¶жҺҘеҸ—еҲ°дәҶ3дёӘеӯҗиҜ·жұӮпјҢе№¶жҢүз…§е“Қеә”ж—¶й—ҙе…ҲеҗҺиҝ”еӣһ

е°Ҹз»“

жүҖд»ҘпјҢжҲ‘д»¬еҸҜд»Ҙеҫ—еҮәд»ҘдёӢз»“и®ә

  1. subrequest жҳҜе№¶еҸ‘иҜ·жұӮзҡ„
  2. еҚідҪҝжҳҜе№¶еҸ‘иҜ·жұӮпјҢеӯҗиҜ·жұӮд№ҹдјҡжҢүз…§еј•е…Ҙзҡ„йЎәеәҸд»Һе…ҲеҲ°еҗҺиҝӣиЎҢиҜ·жұӮгҖӮ

жҖ»з»“

SSIеңЁе®һйҷ…еә”з”ЁдёӯеҸҜд»Ҙеё®жҲ‘д»¬и§ЈеҶідёҖдәӣйЎөйқўеҶ…е®№е…ұз”Ёзҡ„й—®йўҳпјҢдҪҶжҳҜ

  1. SSIжҠҖжңҜдјҡи®©з”ЁжҲ·еҫ—еҲ°иҜ·жұӮзҡ„ж—¶й—ҙжӣҙй•ҝпјҲзҲ¶иҜ·жұӮж—¶й—ҙ + йЎөйқўи§Јжһҗж—¶й—ҙ + SSI иҜ·жұӮж—¶й—ҙ пјү
  2. SSI subrequestдјҡеҚ з”Ёworker connectionж•°йҮҸпјҢйңҖиҰҒжҲ‘д»¬еҜ№д»ЈзҗҶжңҚеҠЎеҷЁзҡ„workerж•°йҮҸиҝӣиЎҢеҗҲзҗҶиҜ„дј°
  3. йҮҮз”ЁSSI жҠҖжңҜпјҢд№ҹдјҡеҜ№upstream serverеёҰжқҘжҲҗеҖҚд»ҘдёҠзҡ„еҺӢеҠӣжҢ‘жҲҳ

жҖ»иҖҢиЁҖд№Ӣ, SSIжҳҜдёҖжҠҠеҸҢеҲғеү‘пјҢеңЁеёҰз»ҷжҲ‘д»¬дҫҝеҲ©зҡ„еҗҢж—¶пјҢд№ҹиҰҒи°Ёж…ҺдҪҝз”ЁпјҒ
